# Create Files

Create a File

```bash
touch file_name
```

Create file in a given path

```bash
touch path_to_file/file_name
```

# Create Directories

**mkdir** = make directory

```bash
mkdir directory_name
```

# Copy files

General Syntax:

```bash
cp [source] [destination]
```

Copy to a directory with same name

```bash
cp file directory/
```

Copy to directory with new name

```bash
cp file directory/new_name
```

# Copy directory

**cp** = copy

General Syntax:

```bash
cp -r [source] [dest]
```

Copy all files from a directory to another directory

```bash
cp -r directory/ BackUpDirectory/
```

# Moving files

General Syntax:

```bash
mv [source] [dest]
```

Move a file to the directory

```bash
mv file_name directory_name/
```

Rename a file

```bash
mv file_name new_file_name
```

Rename a directory

```bash
mv directory/ new_directory/
```

# Delete files

**rm** = remove

Delete a file

```bash
rm file_name
```

Delete a directory

```bash
rm -r directory/
```
