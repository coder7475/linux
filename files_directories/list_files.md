# List Files

List all files

```bash
ls
```

List all files including hidden files

```bash
ls -a
```

Show all files in long list format of directory

```bash
ls -l directory
```

Show all files(including hidden) in long list format

```bash
ls -al
```

Human readable sizes with long list format

```bash
ls -hl
```

All flag combined

```bash
ls -alh
```
