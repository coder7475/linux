# General Commands

**List the users of the OS**:

```bash
   who
```

**Find the current user**:

```bash
  whoami
```

**See the uptime of the server**:

```bash
    uptime
``

**Change the computer hostname**:
1. Access the hostname file

```bash
  sudo nano /etc/hostname
```

2. Edit the file change the host name:

```bash
  sudo nano /etc/hosts
```

3. Reboot the system

```bash
  sudo reboot
```
